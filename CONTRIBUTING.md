# Kako doprinijeti

Hvala na interesu za doprinos! Ovaj projekt je otvoren za sve koji rade s hrvatskim eRačunima.

## Rasprava (Discussions)

Za pitanja, prijedloge i raspravu koristite **GitHub Discussions**:
- **Pitanja** — Ako nešto nije jasno u dokumentaciji
- **Prijedlozi** — Novi primjeri, slučajevi koje nismo pokrili, ispravke
- **Iskustva** — Podijelite kako vaš ERP rješava ove scenarije

## Prijava greške (Issues)

Ako pronađete grešku u dokumentaciji (krivi članak zakona, netočan XML primjer, itd.):
1. Otvorite **Issue**
2. Opišite što je krivo i što bi trebalo pisati
3. Ako možete, navedite izvor (članak zakona, schematron pravilo, specifikacija)

## Pull Requests

1. Forkajte repo
2. Napravite promjene
3. Otvorite Pull Request s opisom što ste promijenili i zašto
4. Navedite izvor za tvrdnje (članak zakona, HR CIUS specifikacija, schematron pravilo)

## Glasanje

Za prijedloge koji zahtijevaju konsenzus zajednice koristite emoji reakcije:
- 👍 Slažem se
- 👎 Ne slažem se
- 🤔 Trebam više informacija

## Pravila

- Budite konstruktivni i profesionalni
- Navodite izvore za tvrdnje o zakonima i propisima
- Hrvatski jezik je preferirani, ali engleski je također prihvatljiv
